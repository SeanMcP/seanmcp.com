Technically this is a garden about typefaces, but I won't be pedantic if you
won't.

Things to look for:

| Letter | Details                    |
| ------ | -------------------------- |
| a      | Double- or single-story    |
| e      | Counter angle and terminal |
| G      | Spur                       |
| g      | Open or loop tail          |
| K/R    | Leg                        |
| S/s    | Curves                     |
| &      | Type and shape             |
